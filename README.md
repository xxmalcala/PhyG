# PhyG
Personal phylogenomic pipeline (PhyG -- "Fig"), incorporates third party tools and includes utility scripts for EDA.


## Overview
Initializing a "pet" project of a new phylogenomic pipeline for exploring evolution across the tree of life.

This will include some prepatory scripts, as well as those I have written for exploratory data analysis.

Use at your own convenience, but updates will be sporadic for the time being.

## Motivation

Many phylogenomic pipelines are limited to their protein/gene-family databases which may or may not be available.
Additionally, many truly "pipe" you from the start to finish, with few manageable options to stop early. This
limited modularity has prompted PhyG ("fig"), which is intended to provide both a "simple" A-Z approach as
well as a more modular suite of steps for the user, along with easily manageable steps for EDA for the less
experienced user.
