# PhyG
In-progress suite of tools and a FAIR principled pipeline for phylogenomic analyses.

## Installation

Clone the repo and move into the directory:
```
git clone https://github.com/xxmalcala/PhyG.git
cd PhyG
```

Installing the PhyG-Pi suite using conda:
```
mamba env create --file env.yml
```

Activate the environment:
```
conda activate phyg_pi
```
